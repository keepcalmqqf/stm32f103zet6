# 安全与敏感信息

本文档描述项目的安全注意事项与敏感信息处理规范。

---

## 当前风险等级

- **低风险**：当前固件为裸机演示程序，不涉及网络通信、用户数据或加密操作。

## 禁止提交的内容

- 私钥、证书、API Key
- 生产环境 Wi-Fi 密码或 MQTT 凭据
- 调试器序列号（若涉及多台设备）
- 个人身份信息或设备唯一标识（若后续接入云端）

## 调试与日志

- `printf` 通过 USART1 输出，任何连接 PA9/PA10 的人都能读取日志。
- 后续若加入敏感信息，应关闭调试日志或进行脱敏处理。

## 硬件安全

- SWD 接口（PA13/PA14）默认开启，便于调试。
- 量产时应考虑：
  - 读保护（RDP）防止固件被提取
  - 禁用 JTAG/SWD 或限制调试访问

## 依赖安全

- HAL 库来自 STM32CubeF1 官方包，保持与 STM32CubeMX 同步更新。
- 第三方代码引入前需评估许可证与来源可信度。
