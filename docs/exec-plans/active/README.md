## 计划模板

新建计划时，建议使用以下结构：

```markdown
# 计划标题

## 目标
一句话描述要达成的结果。

## 背景
为什么要做这件事？参考哪些文档？

## 步骤
1. ...
2. ...

## 决策日志
| 日期 | 决策 | 理由 |
|------|------|------|
| YYYY-MM-DD | ... | ... |

## 验收标准
- [ ] 标准 1
- [ ] 标准 2
```

## 完成后的处理

计划完成后，将其移动到 `docs/exec-plans/completed/`，并在此 README 中移除条目。
