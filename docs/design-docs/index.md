## 如何新增设计文档

1. 在本目录下创建新的 `.md` 文件。
2. 在上方表格中登记文档名称、状态与一句话说明。
3. 如果该决策影响 `AGENTS.md` 中的流程或命令，同步更新 `AGENTS.md`。
4. 在 `ARCHITECTURE.md` 或相关文档中交叉链接。

## 验证状态定义

- **✅ 已验证**：文档内容与当前代码一致，经人工或构建验证。
- **🔄 待验证**：文档已编写，但尚未与代码对齐或验证。
- **⚠️ 已过时**：文档部分内容与当前实现不一致，需要修订。
